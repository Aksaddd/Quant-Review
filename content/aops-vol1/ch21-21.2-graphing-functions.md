Graphing Functions

A function is graphed in the same way as a normal equation is. On the horizontal axis we place the independent variable (x in this case), which can vary freely. On the vertical is the value of the function. The graph of Equation 21.1 is thus the same as if we were graphing the line equation y = |x + 3, with the difference that the vertical axis is called f(x), rather than y. A common test to check if a graph represents a function is that if any vertical line passes through more than one point of the curve, the curve does not represent a function.
