Mathematical Induction

Mathematical induction is a powerful tool when we are asked to prove something is true for integers. It works like this. Suppose we are asked to prove that a given assertion is true for all positive integers. First, we show that it is true for 1 (or some other base case, often 0). Second, we show that if it is true for some integer k, then it must be true for the number k + 1. This is the inductive step. Having proved this we argue that, since it is true for 1, it must be true for 1 + 1 = 2. Since it is true for 2, it is true for 2 + 1 = 3 , and so on.
