## Role Overview

Owns the **emotional and motivational layer** of the platform. Working from McGonigal's four intrinsic rewards framework — satisfying work, hope of success, social connectivity, epic meaning — this role designs every fiero moment, XP system, leveling mechanic, and feedback loop.

Also responsible for the **core UX**: focus mode, cognitive load-aware layouts, and the distraction-free reading experience Dehaene's research demands.

> Must have shipped gamified products before — not a generalist designer.

### McGonigal's Four Intrinsic Rewards
| Reward | Platform Expression |
|--------|---------------------|
| Satisfying work | XP for hard-problem solves, progress bars |
| Hope of success | Fiero moments, "Fun Failure" framing |
| Social connectivity | Leaderboard, study group features |
| Epic meaning | Percentile rank, collective milestones |

---

## Core Responsibilities

- Design the **XP and leveling system**: define XP weights for problem attempts, hint-free solves, spaced review completions
- Create **fiero moment** animations and feedback for hard-problem breakthroughs (post-adversity celebration)
- Design the **"Fun Failure"** feedback system — show closeness, correct technique, and reframe failure as progress
- Build the **Epic Meaning layer**: leaderboard, percentile rank among prep candidates, collective milestones
- Own **focus mode UX**: no sidebar, no notification badges, zero peripheral distractions
- Design session-length nudge and "come back tomorrow" UX flows
- Create the **Personal Notes and Progressive Summarization UI** (Forte's CODE method)
- Design the **Technique Atlas UI** — cross-cutting problem index, navigable by technique type
